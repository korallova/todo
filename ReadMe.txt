ReadMe

Консольное приложение для ведения списка задач - todo.

Список задач сохраняется в файле todo.txt

Приложение поддерживает следующие команды:

    todo.py - показать незавершенные задачи

    todo.py all - показать все задачи (включая завершенные)

    todo.py add <task description> [due:<date>] - добавить задачу с описанием <task description> (и дополнительно можно добавить срок выполнения <date>)

    todo.py remove <n> - удалить задачу с номером <n>. Подтверждение удаления <y>, отмена удаления <n>

    todo.py edit <n> <new task description> - редактировать задачу с номером <n>: установить новое описание <new task description>

    todo.py due <n> <date> - установить срок выполнения либо изменить срок выполнения <date> для задачи <n>

    todo.py done <n> - пометить задачу как выполненную, завершенная задача помечается латинской буквой x в начале строки, и перемещается в конец

    todo.py undo <n> - снять отметку о выполнении

    todo.py search <sting> - поиск задач по тексту <string>

    todo.py created between <date> <date>  - поиск по дате создания между указанными датами

    todo.py created before <date> - поиск по дате создания до указанной даты

    todo.py created after <date> - поиск по дате создания после указанной даты

    todo.py due between <date> <date>  - поиск по дате выполнения между указанными датами

    todo.py due before <date> - поиск по дате выполнения до указанной даты

    todo.py due after <date> - поиск по дате выполнения после указанной даты
